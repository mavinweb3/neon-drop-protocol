# 🟢 NEON DROP PROTOCOL

**Neon Drop Protocol** is a high-fidelity, cyberpunk Web3 minting experience. Built with Next.js 15, GSAP, and thirdweb v5, it features a Global Conduit system—a persistent scroll-synced power line that ignites UI components, roadmap cards, and cryptographic terminals with mathematical precision. Awwwards-tier animations meet zero-trust security.
